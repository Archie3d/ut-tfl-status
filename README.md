## Build
Install [clickable](https://clickable-ut.dev/en/latest/install.html). Then run it with a device connected to build and install:
```bash
clickable
```

To test on a desktop:
```bash
clickable desktop
```

Edit with QtCreator:
```bash
clickable ide
```
